  ```typescript

    stamps: CollectionStamp[];
    finds: CollectionStamp[];
    discoveries: CollectionStamp[];
    onViewMore: (type: 'stamp' | 'find' | 'discovery') => void;
    activeTabIndex: SharedValue<number>;  // shared with OrbBackground
    onTabChange: (index: number) => void; // for React state sync
  }
  ```

  **Tab bar (above PagerView):**
  - 3 labels: "Stamps", "Finds", "Discoveries" in a row, each `screenWidth / 3` wide
  - Frosted glass pill behind active tab: Animated.View positioned absolutely, width = TAB_WIDTH, contains `<BlurView intensity={30} tint={isDark?'dark':'light'} blurMethod="dimezisBlurViewSdk31Plus" style={{ flex:1, borderRadius: 20, overflow:'hidden' }}>` with inner tint overlay `rgba(255,255,255,0.08)` dark / `rgba(0,0,0,0.04)` light
  - Pill translateX driven by `tabOffset` shared value (updated from `onPageScroll` for smooth mid-swipe animation)
  - Tab labels: active = `colors.text` Poppins_600SemiBold, inactive = `colors.textSecondary` Poppins_500Medium, 14px
  - Tab press calls `pagerRef.current?.setPage(index)`

  **PagerView:**
  - `initialPage={0}`, `style={{ flex: 1 }}`
  - `onPageSelected` → update React state via `onTabChange(position)` AND `activeTabIndex.value = position`
  - `onPageScroll` → `tabOffset.value = position + offset` (smooth pill tracking)
  - Each page child has `key="0"`, `key="1"`, `key="2"` (required by PagerView)
  - Page 0: `<ScrollView><GlassGrid items={stamps} type="stamp" onViewMore={...} /></ScrollView>`
  - Page 1: `<ScrollView><GlassGrid items={finds} type="find" onViewMore={...} /></ScrollView>`
  - Page 2: `<ScrollView><GlassGrid items={discoveries} type="discovery" onViewMore={...} /></ScrollView>`
  - Each ScrollView has `contentContainerStyle={{ padding: 16, paddingBottom: 120 }}` (bottom padding for tab bar)

  Use both `useState` for React re-renders AND `useSharedValue` for smooth pill animation without re-renders (synced in onPageSelected). See research Anti-Patterns section.
