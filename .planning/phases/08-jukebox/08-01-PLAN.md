   ```sql
   CREATE TABLE IF NOT EXISTS event_artists (
     id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
     event_id UUID NOT NULL REFERENCES events(id) ON DELETE CASCADE,
     performer_id UUID NOT NULL REFERENCES performers(id) ON DELETE CASCADE,
     sort_order INT DEFAULT 0,
     created_at TIMESTAMPTZ DEFAULT now(),
     UNIQUE(event_id, performer_id)
   );
   ```
   Use `psql` or Supabase CLI to run this. Verify with a SELECT on the table.

   export type JukeboxItem = {
     id: string;              // collection.id

3. **GET /api/mobile/jukebox endpoint** at `~/decibel/src/app/api/mobile/jukebox/route.ts`:
   - Copy `getAuthEmail` + admin client pattern from discover/route.ts
   - Accept query param `page` (default 0), page size 20
   - Step 1: Get fan by email
   - Step 2: Get fan's following IDs from `fan_follows` where `follower_id = fan.id`
   - Step 3: Query `collections` JOIN `fans` JOIN `performers` where:
     - `collection_type = 'find'`
     - `created_at >= now()-48h`
     - `fan_id IN followingIds` (if following list non-empty)
     - Order by `created_at DESC`
     - Range: `page*20` to `(page+1)*20-1`
   - Step 4: If followingIds was non-empty but result is empty, re-query WITHOUT the `fan_id` filter (fallback). Set `isFallback = true`.
   - Step 5: For each item, derive embed URL if stored embed columns are null:
     - Spotify: `https://open.spotify.com/artist/ABC` -> `https://open.spotify.com/embed/artist/ABC`
     - SoundCloud: `https://soundcloud.com/slug` -> `https://w.soundcloud.com/player/?url=https://soundcloud.com/slug&auto_play=false&visual=true`
     - Apple Music: `https://music.apple.com/us/artist/name/123` -> `https://embed.music.apple.com/us/artist/name/123`
   - Step 6: Fire-and-forget upsert of derived embed URLs back to `performers` table (don't await, don't block response)
   - Step 7: Determine `platform` from whichever URL exists (spotify_url -> 'spotify', soundcloud_url -> 'soundcloud', apple_music_url -> 'apple_music')
   - Return `{ items: JukeboxItem[], hasNextPage: boolean, isFallback: boolean }`
