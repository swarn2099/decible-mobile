- Render venue name as `<SvgText>` centered horizontally near the top of the circle (y = size * 0.3), uppercase, Poppins-SemiBold, fontSize=8, fill="#FF4D6A"
- Truncate venue name to 18 chars max with "..." if longer

Date in center:
- `<SvgText>` at center (y = size * 0.52), monospace font (`Platform.OS === "ios" ? "Courier" : "monospace"`), fontSize=10, fill="#FF4D6A", fontWeight="bold"
- Format: "MAR 11, 2026" (abbreviated month, day, year from event_date)

Artist name below center:
- `<SvgText>` at (y = size * 0.68), Poppins-Medium, fontSize=7, fill="#FF4D6A", opacity=0.8
- If multiple artists (stamp has venue with lineup), show first artist name, truncate to 16 chars

Overall stamp opacity: 0.85-0.95 (subtle faded ink effect). Use `opacity` on the wrapping View.

**Dark mode glow (only when isDark):**
