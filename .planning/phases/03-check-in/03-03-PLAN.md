**Build StampAnimationModal:**

`src/components/checkin/StampAnimationModal.tsx` — Full-screen modal overlay (ANIM-01, ANIM-02, ANIM-03):

Props: `{ visible: boolean, stamps: StampData[], onViewPassport: () => void, onDismiss: () => void }`

Implementation:
- React Native `Modal` with `transparent={true}`, `animationType="fade"`
- Dark semi-transparent background with `BlurView` (expo-blur, intensity 25)
- Center of screen: stamp animation area

**Animation sequence (adapt from ConfirmationModal pattern):**
1. Stamp starts off-screen (translateY = -300, scale = 1.2)
2. Stamp slams down to center with `withSpring(0, { damping: 12, stiffness: 180 })` — on finish, trigger haptic (ANIM-02): `Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Medium)`
3. If Lottie file exists and works: play `LottieView` with `autoPlay={false}`, `loop={false}`, trigger `ref.current?.play()` when modal becomes visible. Use `colorFilters` to tint to #FF4D6A. `onAnimationFinish` triggers the reveal. If Lottie doesn't work, use the Reanimated stamp-slam from ConfirmationModal as fallback.
4. Ink spread effect: circular View expanding from center with opacity fade (Reanimated `withTiming` scale 0->3, opacity 0.6->0 over 400ms) — pink tint (#FF4D6A at 30% opacity) (ANIM-03)
5. After slam + ink spread (~400ms), fade in reveal text with `withDelay(400, withTiming(1, { duration: 300 }))`:
   - Venue name: largest text, Poppins Bold 24px, white
   - Date: monospace style (use Platform.select for monospace font or Poppins with letter-spacing), 14px, white at 70% opacity, format as "Mar 11, 2026"
   - Artist name(s): Poppins SemiBold 16px, white, listed vertically if multiple
6. After text (~700ms), fade in action buttons:
   - Primary: "View Passport" — pink background, white text, navigates to Passport tab (router.push('/(tabs)/passport'))
   - Secondary: "Done" — transparent, white text, calls onDismiss

For multi-artist lineups (Scenario A): single stamp animation, all artist names listed vertically in the reveal (locked decision).
